# 💊 Pharmacy Sales Performance Dashboard

## 📊 Project Overview

This project presents a **Power BI dashboard** built to analyze pharmacy sales data.
It provides insights into **revenue, profit, sales trends, and customer behavior** using interactive visualizations.

---

## 🎯 Objectives

* Analyze overall **sales and revenue performance**
* Identify **top-selling medicines and categories**
* Track **monthly sales trends**
* Understand **customer payment preferences**
* Evaluate **profitability (profit & margin)**

---

## 📁 Dataset

* Source: Excel file (`Cleaned_Data`)
* Contains:

  * Date
  * Medicine Name
  * Category
  * City
  * Quantity
  * Cost Price
  * Final Amount
  * Payment Mode

---

## 🧮 Key Metrics (DAX Measures)

* **Total Sales**
* **Total Revenue**
* **Total Quantity**
* **Total Profit**
* **Profit Margin %**

---

## 📊 Dashboard Features

* 🔝 KPI Cards (Sales, Revenue, Quantity, Profit Margin)
* 📈 Revenue Trend (Monthly)
* 💊 Sales by Category
* 🧪 Top Medicines Analysis
* 📍 Revenue by City (Map)
* 💳 Payment Mode Distribution
* 🎛 Interactive Filters (Date, City, Category)

---

## 🛠 Tools & Technologies

* Microsoft Power BI

* DAX (Data Analysis Expressions)
